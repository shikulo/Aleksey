 4. Вывести адрес приложения todolist в системе Android
 C:\Users\user>adb shell pm list "packages | grep todolist"
package:com.android.todolist

 5. Удалить приложение todolist с телефона через ADB
 C:\Users\user>adb shell pm uninstall com.android.todolist
Success

 6. Сделать скриншот.
C:\Users\user>adb shell screencap /sdcard/screen.png

 7. Скопировать скриншот на компьютер.
C:\Users\user>adb pull /sdcard/screen.png /Users/user/Pictures/screen_group23.png
/sdcard/screen.png: 1 file pulled, 0 skipped. 7.4 MB/s (516336 bytes in 0.067s)

 8. Сделать скриншот и сразу скопировать на компьютер в одной команде.
 C:\Users\user>adb shell screencap /sdcard/screen1.png && adb pull /sdcard/screen1.png /Users/user/Pictures/screen_group23.png
 
 9. Вывести в консоль логи приложения todolist:
 C:\Users\user>adb logcat
 -что бы остановить комбинация Ctrl+C
 отфильтруем логи нашего приложения todolist:
 C:\Users\user>adb logcat | findstr -rnw "todolist"
 -проработаем нужные логи и сохраним на компьютер:
 C:\Users\user>adb logcat | findstr -rnw "todolist" > log_todolist.log
 обнулить буффер логов:
 C:\Users\user>adb logcat -c
 
 10. Backup:
 C:\Users\user>adb backup -apk -shared -all -f /Users/user/Pictures/backup.ad 
  C:\Users\user>adb restore /Users/user/Pictures/backup.ad
